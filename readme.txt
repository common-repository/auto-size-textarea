Auto Size Textarea is a simple plugin to auto size the textarea of your wordpress website. 

== Description ==
This is a very simple plugin to auto-size textarea. Just install and activate this plugin. It will automatically auto-size all the textarea of the website. 

== Installation ==
1. Upload the `bc-autosize` folder to the `/wp-content/plugins/` directory via FTP.
2. Activate the plugin.
